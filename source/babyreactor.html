babyreactor.flux.iterate_flux(flux_old, matrix, k_old, fission)[source]¶

The next iteration of the flux

Produces the next iteration of the flux using the method on page 5-279 of Deuderstadt and Hamilton’s book.

Parameters:
  • flux (numpy.ndarray) – Array containing the flux value at each node
  • matrix (numpy.ndarray) – 2-d array with the removal operator
  • k (float) – Neutron multiplication constant for the reactor
  • fission (numpy.ndarray) – Fission constants for each node
Returns:

next flux iteration, next k iteration, error in the flux (really deviation between generations), generational change in k
Return type:

tuple
