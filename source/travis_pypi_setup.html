travis_pypi_setup module¶

Update encrypted deploy password in Travis config file

travis_pypi_setup.encrypt(pubkey, password)[source]¶

Encrypt password using given RSA public key and encode it with base64.

The encrypted password can only be decrypted by someone with the private key (in this case, only Travis).

travis_pypi_setup.fetch_public_key(repo)[source]¶

Download RSA public key Travis will use for this repo.

Travis API docs: http://docs.travis-ci.com/api/#repository-keys

travis_pypi_setup.load_key(pubkey)[source]¶

Load public RSA key, with work-around for keys using incorrect header/footer format.

Read more about RSA encryption with cryptography: https://cryptography.io/latest/hazmat/primitives/asymmetric/rsa/
